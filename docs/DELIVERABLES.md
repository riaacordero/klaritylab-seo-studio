## 🚀 How to Use

### Installation & Setup
```bash
# Navigate to project
cd /Users/riaacordero/Documents/Dev/klaritylab-seo-studio

# Install dependencies (if needed)
npm install

# Configure API keys
# Create .env.local with:
# GOOGLE_GEMINI_API_KEY=your_key
# OR OPENAI_API_KEY=your_key
```

### Running Development
```bash
# Start dev server
npm run dev

# Navigate to http://localhost:3000
# Interlink Review tab will be the default view
```

### Building for Production
```bash
# Production build
npm run build

# Type checking
npx tsc --noEmit

# Output is in .next/ directory
```

### Testing the Feature
1. Click "Interlink Review" tab (should be default)
2. Fill in 5 required URL fields
3. (Optional) Add related blog pages
4. Click "Analyze Interlinking"
5. Review results table
6. Click "Copy Table" or "Export CSV"
