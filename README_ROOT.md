## 🚀 Quick Start

### 1. Start Development Server
```bash
npm run dev
```
Opens on http://localhost:3000

### 2. Test the Interlink Review Tab
- Click the "Interlink Review" tab (default view)
- Fill in the required URL fields
- Click "Analyze Interlinking"
- Test Copy Table or Export CSV buttons

### 3. Build for Production
```bash
npm run build
```
