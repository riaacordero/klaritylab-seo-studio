================================================================================
STEP 1: GET YOUR FREE GEMINI API KEY (2 minutes)
================================================================================

1. Visit: https://aistudio.google.com/app/apikey
2. Click the "Create API Key" button
3. Copy your API key (it starts with "AIzaSy...")
4. Keep it safe - you'll need it in Step 2

(It's FREE! No credit card required for development and testing)

================================================================================
STEP 2: CONFIGURE YOUR PROJECT (1 minute)
================================================================================

In your terminal, run these commands:

    cd /Users/riaacordero/Documents/Dev/klaritylab-seo-studio
    cp .env.example .env.local

Then open .env.local and paste your API key:
    
    GEMINI_API_KEY=your_api_key_here_paste_after_equals

Save the file and you're done!

================================================================================
STEP 3: INSTALL & START (1 minute)
================================================================================

In the same terminal, run:

    npm install
    npm run dev

You'll see output like:
    ▲ Next.js 14.2.35
    - Local:        http://localhost:3000
    ✓ Ready in 1889ms

Copy that URL!

================================================================================
STEP 4: TRY IT OUT (1 minute)
================================================================================

1. Open http://localhost:3000 in your browser
2. You'll see a beautiful form asking for your SEO question
3. Try this prompt: "Create an interlinking strategy for my SaaS blog"
4. Click "Get AI Recommendations"
5. Watch the magic happen! ✨

That's it! You now have a working AI agent!

================================================================================
TROUBLESHOOTING
================================================================================

"Provider not configured" error?
    → Check that .env.local exists
    → Check that GEMINI_API_KEY is correctly pasted
    → Restart npm run dev after editing .env.local

"Module not found" error?
    → Run: npm install
    → Then: npm run dev

Dev server won't start?
    → Check that port 3000 is not in use
    → Try: npx next dev -p 3001 (to use port 3001 instead)

Still stuck?
    → Check the README.md troubleshooting section
    → Review QUICK_START.md for detailed setup steps
